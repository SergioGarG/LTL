# LTL

How to use it:

Adds the following dependency and repository in your POM.xml

```
	<repositories>
		<repository>
			<id>LTL</id>
			<url>https://github.com/co4robots</url>
		</repository>
	</repositories>


		<dependency>
			<groupId>se.gu</groupId>
			<artifactId>LTL</artifactId>
			<version>1.0.0</version>
		</dependency>
```
